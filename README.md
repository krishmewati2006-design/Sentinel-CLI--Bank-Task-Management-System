# Sentinel CLI: Bank & Task Management System

### 🎯 Overview
A Command Line Interface (CLI) application built to demonstrate core Object-Oriented Programming (OOP) principles. This system manages user accounts, secure financial transactions, and task assignment using a robust backend-first logic.

### 🛠️ Tech Stack
* **Language:** JavaScript (Node.js)
* **Architecture:** Object-Oriented Programming (OOP)
* **Concepts:** Encapsulation, Inheritance, Data Validation

### ✨ Key Features
* [cite_start]**Secure Data (Encapsulation):** Uses private class fields (`#balance`) to prevent unauthorized direct access to sensitive financial data[cite: 35].
* [cite_start]**Role-Based Access (Inheritance):** Features a base `User` class and an `Admin` class with elevated permissions[cite: 35].
* [cite_start]**Safe Transactions:** Integrated validation logic to prevent negative deposits or overdrafts[cite: 49].
* [cite_start]**Task Management:** Dynamic task tracking using modern Spread and Destructuring syntax[cite: 34].
